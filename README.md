# Duolingo Case Study

A single-page static case study built for GitHub Pages.

## Structure

- `index.html` - the full site, including markup, styles, and script

## Host on GitHub Pages

1. Create a GitHub repository and upload these files.
2. Make sure `index.html` is in the repository root.
3. In GitHub, go to **Settings** > **Pages**.
4. Set the source to the default branch and the root folder.
5. Save and wait for GitHub Pages to publish the site.

## Notes

- The page is fully static, so it does not need a build step.
- It uses Google Fonts from a CDN, so an internet connection is required for the intended typography.
